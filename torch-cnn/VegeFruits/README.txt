1. Download weight file
  Please run the following command on your Anacond Prompt to download a weight file.

 >python download_weight_file.py


2. If you would like to build your own weight file by your hands, please run the following commands
on your Anacond Prompt.
 
 >python TorchVegeFruitsDataSetAugmentor.py
 
 >python TorchVegeFruitsModel.py


3. If you would like to classify images by TorchVegeFruitsClassifier, please run the following commands
on your Anacond Prompt.
 
 >python TorchVegeFruitsClassifier.py
